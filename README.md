# AstrBot 聊天收费插件

为 AstrBot 提供聊天收费能力，支持**按次计费**和**订阅制**两种模式，并区分**私聊**与**群聊**独立计费。

---

## 功能特性

- 🔹 **按次计费**：每条 AI 消息消耗积分，新用户免费试用 N 条
- 🔹 **订阅制**：包天/包月/包年，到期自动拦截
- 🔹 **私聊 & 群聊独立**：个人充值和群聊充值分离，群聊支持公共余额/群订阅
- 🔹 **扣费优先级**：个人余额 > 群公共余额（可避免群聊过度消耗个人积分）
- 🔹 **并发安全**：每个作用域（私聊用户/群）独立异步锁
- 🔹 **旧版数据兼容**：自动迁移旧版 key，无需手动操作
- 🔹 **WebUI 配置面板**：所有价格、文案、白名单均可通过 WebUI 修改
- 🔹 **自定义文案**：10 项提示文案支持 `{balance}` `{price}` 等占位符

---

**要求**：AstrBot >= 4.9.2

## 配置

所有配置在 **WebUI → 插件管理 → 聊天收费 → 配置面板** 中修改。

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `mode` | 收费模式：`per_msg`（按条）/ `subscription`（订阅制） | `per_msg` |
| `price_per_msg` | 每条消息价格（积分） | `1` |
| `free_trial_count` | 新用户免费试用次数 | `10` |
| `subscribe_prices` | 订阅套餐列表（`template_list`） | 7天/10, 30天/30, 365天/299 |
| `admin_ids` | 管理员 ID 列表 | `[]` |
| `whitelist_ids` | 白名单用户 ID 列表 | `[]` |
| `tpl_balance_short` | 余额不足提示文案 | 见代码 |
| `tpl_balance_info` | 余额查询回复 | 见代码 |
| `tpl_group_balance_info` | 群余额查询回复 | 见代码 |
| `tpl_sub_expired` | 订阅过期提示（个人） | 见代码 |
| `tpl_group_sub_expired` | 订阅过期提示（群） | 见代码 |
| `tpl_sub_info` | 订阅状态回复（个人） | 见代码 |
| `tpl_group_sub_info` | 订阅状态回复（群） | 见代码 |
| `tpl_charge_success` | 充值成功提示 | 见代码 |
| `tpl_deduct_success` | 扣费成功提示 | 见代码 |
| `tpl_sub_add_success` | 添加订阅成功 | 见代码 |

## 指令说明

### 用户指令

| 命令 | 说明 | 示例 |
|------|------|------|
| `/balance` | 查询余额/订阅状态（群聊中同时显示个人+群信息） | `/balance` |
| `/收费价格` | 查询当前计费标准和订阅套餐 | `/收费价格` |

### 管理员指令（个人）

| 命令 | 说明 | 示例 |
|------|------|------|
| `/充值 用户ID 金额` | 给个人充值积分 | `/充值 123456 100` |
| `/扣费 用户ID 金额` | 扣除个人积分 | `/扣费 123456 50` |
| `/添加订阅 用户ID 天数` | 给个人添加订阅天数 | `/添加订阅 123456 30` |

### 管理员指令（群聊）

| 命令 | 说明 | 示例 |
|------|------|------|
| `/群充值 群ID 金额` | 给群充值公共积分 | `/群充值 789012 100` |
| `/群扣费 群ID 金额` | 扣除群公共积分 | `/群扣费 789012 50` |
| `/群添加订阅 群ID 天数` | 给群添加订阅天数 | `/群添加订阅 789012 30` |

### 管理员指令（统计）

| 命令 | 说明 | 示例 |
|------|------|------|
| `/查看统计` | 查看私聊/群聊的余额和订阅统计 | `/查看统计` |

## 扣费逻辑

### per_msg 模式
```
用户发消息 → 检查免费次数 → 个人余额 → 群公共余额 → 拦截
       放行✅     放行✅     扣个人✅   扣群✅      拦截❌
```

### subscription 模式
```
用户发消息 → 个人订阅未过期 → 群订阅未过期 → 拦截
        放行✅       放行✅       拦截❌
```
